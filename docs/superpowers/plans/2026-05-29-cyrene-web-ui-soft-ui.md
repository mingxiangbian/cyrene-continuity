# Cyrene Web UI Soft UI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build `cyrene-continuity codex ui`, a local Warm Cream Coral Soft UI console for read-only memory pipeline visibility plus project harvester dry-run preview.

**Architecture:** Add a small Node `http` server behind the existing Codex CLI router. API code is isolated from HTTP routing, static UI assets live in `src/ui/static`, and `scripts/generate-ui-static.mjs` generates an inline TS asset module so the bundled plugin runtime can serve the UI without reading source files.

**Tech Stack:** Node 20 built-ins (`http`, `fs/promises`, `net`), TypeScript, existing Cyrene memory readers, plain HTML/CSS/JS, Vitest, Browser verification.

---

## File Structure

- `scripts/generate-ui-static.mjs`: reads `src/ui/static/index.html`, `app.js`, and `styles.css`; writes generated `src/codex/codex-ui-static.generated.ts`.
- `scripts/build-plugin.mjs`: runs the UI static generator before esbuild.
- `src/codex/codex-ui-static.ts`: small wrapper around generated static assets with MIME types.
- `src/codex/codex-ui-static.generated.ts`: generated asset map committed to keep `npm run typecheck` working from a clean checkout.
- `src/codex/codex-ui-api.ts`: pure read/dry-run API dispatcher for `/api/*`, no HTTP concerns.
- `src/codex/codex-ui-server.ts`: local-only HTTP server, static routing, JSON routing, lifecycle/close helpers.
- `src/codex/codex-cli.ts`: adds `codex ui [--port <n>]`.
- `src/ui/static/index.html`: app shell only, no inline application logic.
- `src/ui/static/app.js`: fetches API endpoints, renders tabs/details, runs dry-run.
- `src/ui/static/styles.css`: Warm Cream Coral Soft UI tokens and responsive Sidebar Console.
- `tests/codex-ui-static.test.ts`: generated/static asset coverage.
- `tests/codex-ui-api.test.ts`: API dispatcher and read/dry-run safety coverage.
- `tests/codex-ui-server.test.ts`: HTTP/static/API/local-only server coverage.
- `tests/codex-cli.test.ts`: CLI usage and `codex ui --port 0` smoke coverage.
- `tests/codex-ui-assets.test.ts`: static UI labels, disabled write actions, palette tokens.
- `README.md`: documents `codex ui` v1 scope.
- `plugin/skills/cyrene-continuity/SKILL.md`: documents when to use Web UI and pending-only boundary.
- `plugin/runtime/cyrene-continuity.mjs`: generated by `npm run build:plugin`.

Create `scripts/generate-ui-static.mjs`:

```js
#!/usr/bin/env node
import { mkdir, readFile, writeFile } from 'node:fs/promises'
import { dirname, resolve } from 'node:path'
import { fileURLToPath } from 'node:url'

const repoRoot = resolve(dirname(fileURLToPath(import.meta.url)), '..')
const staticRoot = resolve(repoRoot, 'src/ui/static')
const outfile = resolve(repoRoot, 'src/codex/codex-ui-static.generated.ts')

const assets = [
  { route: '/', file: 'index.html', contentType: 'text/html; charset=utf-8' },
  { route: '/app.js', file: 'app.js', contentType: 'text/javascript; charset=utf-8' },
  { route: '/styles.css', file: 'styles.css', contentType: 'text/css; charset=utf-8' }
]

const entries = []
for (const asset of assets) {
  const body = await readFile(resolve(staticRoot, asset.file), 'utf8')
  entries.push(`  ${JSON.stringify(asset.route)}: { contentType: ${JSON.stringify(asset.contentType)}, body: ${JSON.stringify(body)} }`)
}

await mkdir(dirname(outfile), { recursive: true })
await writeFile(
  outfile,
  [
    '// Generated by scripts/generate-ui-static.mjs. Do not edit manually.',
    '',
    'export interface CodexUiStaticAsset {',
    '  contentType: string',
    '  body: string',
    '}',
    '',
    'export const CODEX_UI_STATIC_ASSETS: Record<string, CodexUiStaticAsset> = {',
    entries.join(',\n'),
    '}',
    ''
  ].join('\n'),
  'utf8'
)
```

```ts
export async function handleCodexUiApiRequest(input: {
  cwd: string
  method: string
  pathname: string
  body?: unknown
  now?: string
  callModel?: typeof defaultCallModel
}): Promise<CodexUiApiResult> {
  try {
    if (input.pathname === '/api/memory/harvest-project/dry-run') {
      if (input.method !== 'POST') return methodNotAllowed()
      const result = await runCodexProjectMemoryHarvest({
        cwd: input.cwd,
        config: createDefaultConfig(input.cwd),
        callModel: input.callModel ?? defaultCallModel,
        dryRun: true,
        now: input.now
      })
      return ok({ result })
    }

    if (input.method !== 'GET') return methodNotAllowed()

    switch (input.pathname) {
      case '/api/status':
        return ok(await readCodexMemoryStatus({ cwd: input.cwd }))
      case '/api/dashboard':
        return ok(await readDashboard(input.cwd, input.now))
      case '/api/memory/pending':
        return ok(await listCodexPendingMemories({ cwd: input.cwd }))
      case '/api/memory/active':
        return ok(await readActive(input.cwd))
      case '/api/review-summaries':
        return ok(await readReviewSummaries(input.cwd))
      case '/api/project-memory':
        return ok(await readGroupedProjectMemory(input.cwd))
      case '/api/dream':
        return ok(await readDream(input.cwd))
      case '/api/profile':
        return ok(await readProfile(input.cwd))
      default:
        return notFound()
    }
  } catch (error) {
    return failure(500, 'internal_error', error instanceof Error ? error.message : String(error))
  }
}
```

Implement helper responses `ok`, `failure`, `notFound`, `methodNotAllowed`.

Helper behavior:

- `readActive(cwd)` identifies project, reads `codexProjectMemoryRoot(projectId)`, returns `{ project, active, memoryRoot }`.

- `readGroupedProjectMemory(cwd)` groups active project memory into exact labels: `Project Facts`, `Project Decisions`, `Workflow Rules`, `Known Pitfalls`, `Rejected Approaches`, `Open Questions`, `Other Project Memory`.   const host = '127.0.0.1' as const
```ts
  const requestedPort = input.port ?? 47833
  const server = createServer((request, response) => {
    void handleRequest({ cwd: input.cwd, request, response })
  })
  const port = await listenWithFallback(server, host, requestedPort)
  return {
    host,
    port,
    url: `http://${host}:${port}`,
    close: () => new Promise((resolve, reject) => server.close((error) => error === undefined ? resolve() : reject(error)))
  }
}
```

Implement private helpers:

- `listenWithFallback(server, host, port)` tries requested port. If port is `0`, use the assigned port. If nonzero port gets `EADDRINUSE`, try up to 20 following ports. Re-throw other errors.
- `handleRequest` parses `new URL(request.url ?? '/', 'http://127.0.0.1')`.
- For `/api/*`, read JSON body only for non-GET requests with a small cap such as 64 KiB, call `handleCodexUiApiRequest`, and send JSON.
- For static paths, serve `/`, `/app.js`, `/styles.css` from `getCodexUiStaticAsset`.
- Unknown static route returns status 404, `text/plain; charset=utf-8`, body `Not found\n`.
- JSON responses use `Cache-Control: no-store`.
- Static responses may use `Cache-Control: no-store` in v1 to simplify development.

```ts
async function readChildStdoutUntil(child: ReturnType<typeof execFile>, pattern: RegExp): Promise<string> {
  return new Promise((resolve, reject) => {
    let output = ''
    const timer = setTimeout(() => reject(new Error(`Timed out waiting for stdout matching ${pattern}: ${output}`)), 10_000)
    child.stdout?.on('data', (chunk: Buffer) => {
      output += chunk.toString('utf8')
      const line = output.split(/\r?\n/).find((item) => pattern.test(item))
      if (line !== undefined) {
        clearTimeout(timer)
        resolve(line)
      }
    })
    child.on('error', (error) => {
      clearTimeout(timer)
      reject(error)
    })
    child.on('exit', (code) => {
      if (!pattern.test(output)) {
        clearTimeout(timer)
        reject(new Error(`Process exited with ${code} before UI URL: ${output}`))
      }
    })
  })
}
```

```ts
function parseOptionalNonNegativeInteger(args: string[], option: string, label: string): number | undefined {
  const hasOption = args.includes(option) || args.some((arg) => arg.startsWith(`${option}=`))
  if (!hasOption) return undefined
  const value = parseOptionalOption(args, option)
  if (value === undefined) {
    throw new Error(`Invalid ${label}: missing value`)
  }
  const parsed = Number(value)
  if (!Number.isInteger(parsed) || parsed < 0 || parsed > 65535) {
    throw new Error(`Invalid ${label}: expected integer port 0-65535`)
  }
  return parsed
}

async function waitForProcessTermination(close: () => Promise<void>): Promise<void> {
  await new Promise<void>((resolve) => {
    const done = (): void => resolve()
    process.once('SIGINT', done)
    process.once('SIGTERM', done)
  })
  await close()
}
```
